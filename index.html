Documentação do Neuraflow

Uma biblioteca de aprendizado de máquina simples para Python

Introdução

O NeuraFlow é uma biblioteca de aprendizado de máquina em Python, baseada em redes neurais e outras técnicas de aprendizado supervisionado e não supervisionado. Ele fornece uma interface simples para treinar e avaliar modelos, além de permitir a manipulação avançada de dados.

A biblioteca utiliza scikit-learn, imblearn e outras bibliotecas populares de aprendizado de máquina para realizar tarefas como classificação, regressão, análise de desempenho, validação cruzada, entre outras.

Instalação

Para instalar o NeuraFlow, basta utilizar o pip para instalar as dependências necessárias:

pip install numpy joblib scikit-learn imbalanced-learn matplotlib

Estrutura da Classe RedeNeural

A classe RedeNeural é o principal componente da biblioteca. Ela contém vários métodos para treinamento, avaliação e análise de modelos de aprendizado de máquina. A classe foi projetada para ser flexível, suportando diferentes tipos de modelos e oferecendo várias ferramentas para análise e melhoria do desempenho.

Atributos

  • camadas: Define o número de unidades em cada camada oculta da rede neural (utilizado pelo MLPClassifier). O valor padrão é [5].
  • max_iter: Define o número máximo de iterações para o treinamento. O valor padrão é 1000.
  • solver: O solver a ser usado no treinamento. O valor padrão é 'adam'.
  • regularizacao: Tipo de regularização a ser utilizada no modelo. Pode ser 'l2' (MLPClassifier) ou 'l1' (LogisticRegression). O valor padrão é 'l2'.
  • C: Parâmetro de regularização que controla a força da penalização, se estiver usando regularização L1 ou L2. O valor padrão é 1.0.
  • model: O modelo de aprendizado de máquina (MLPClassifier ou LogisticRegression).
  • historia: Histórico da curva de perda durante o treinamento.

Exemplos de Uso

Exemplo 1: Treinamento de uma Rede Neural

from neuraflow import RedeNeural
rede = RedeNeural(camadas=[5, 3])
rede.treinar(X_train, y_train)

Exemplo 2: Avaliação do Modelo

precisao = rede.testar(X_test, y_test)
print(f"Acurácia: {precisao * 100:.2f}%")

Exemplo 3: Análise de Importância das Características

rede.analise_importancia(X_train)

Exemplo 4: Validação Cruzada

rede.cross_validation(X_train, y_train, k=5)
